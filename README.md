# Tic-Tac-Toe

A console-based Tic-Tac-Toe game built in C#.
This project was created to practise object-oriented programming, game state management, user input validation and gameplay logic.

## Features
- Two-player turn-based gameplay
- Board display using a 3x3 grid
- Win detection for rows, columns and diagonals
- Draw detection when the board is full
- Input validation to prevent players selecting occupied spaces
- Board updates after each player move

## Technologies Used
- C#
- .NET
- Console Application

## What I Learned

Through this project I gained experience working with classes, methods, arrays, loops and conditional logic. I also developed my understanding of object-oriented design by separating responsibilities between the game manager, player and board systems.
Building the win detection and input validation systems helped strengthen my problem-solving and debugging skills while developing a complete playable application.
